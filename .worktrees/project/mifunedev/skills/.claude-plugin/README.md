# `.claude-plugin/` — Claude Code marketplace mirror (reserved for V1)

This directory is the landing spot for `marketplace.json`, the Claude Code plugin marketplace mirror of `registry.json`. It is **intentionally empty in V0**.

## Contents

| File | Purpose |
|------|---------|
| `marketplace.json` | (V1) Claude Code marketplace descriptor generated by `scripts/publish-registry.sh` from `registry.json`. Not present in V0. |

## Conventions

- This directory is populated by `scripts/publish-registry.sh` — do not hand-edit `marketplace.json`.
- V1 will write `marketplace.json` here as part of the registry publish step.
- Non-Claude-Code consumers ignore this directory entirely.
