# progress

## Codebase Patterns

- `gh issue create` needs the repo context: run it from the repo root with an
  explicit `--repo <owner/name>` (running from a non-repo cwd like /tmp makes it
  fail "not a git repository"). Capture the new number with `grep -oE '[0-9]+$'`
  on the printed URL.
- Docusaurus parses `.md` as MDX. Bare `<...>` AND bare `{...}` break the build —
  wrap set-notation like `{skills, agents}` in backticks. The docs sidebar is
  frontmatter-driven (`sidebar_position`); no manual sidebar registration.
- M0 is the roadmap surface only — no migration. The migrations are M1–M6.
