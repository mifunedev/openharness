### Learnings for future iterations
- The `**` glob in YAML paths is not a YAML anchor — grep for actual YAML anchor syntax `&name` / `*name` separately.
- GitHub Actions does not support YAML anchors, so paths lists must be explicitly duplicated between push and pull_request.
- Workflow-level concurrency block (sibling of on:/jobs:) is different from job-level placement (inside jobs.ci:).
