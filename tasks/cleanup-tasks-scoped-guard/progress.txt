## Codebase Patterns

- **Pre-commit hook** (`.husky/pre-commit`) runs `pnpm run lint && pnpm run test`.
  `lint` is a recursive `--if-present` no-op (no per-package lint scripts), but
  `test` runs the full vitest suite (186 tests, ~3s). A commit will fail if any
  test breaks, so keep changes test-clean. There is no markdownlint/prettier in
  the hook â€” markdown edits only need to keep the test suite green.
- **No bare `git status --porcelain`**: the US-003 probe will subtract scoped
  lines via `grep -v -- '-- tasks/'` and fail on any survivor. Keep the scoped
  form `git status --porcelain -- tasks/ ':!tasks/archive/'` as the ONLY
  occurrence of that substring in `crons/cleanup-tasks.md` â€” never reference a
  bare `git status --porcelain` in prose.
- **Probe template** `evals/probes/owned-surface-guard.sh`: static grep oracle,
  3-state (PASS=0/REGRESSION=1/SKIPPED=2), uses `grep -F` for literal patterns
  with shell metacharacters and `# tier:`/`# source:`/`# desc:` headers +
  `set -euo pipefail`. Mirror it for US-003.
- **Liveness tokens** in `crons/cleanup-tasks.md`: `OK (archived N, skipped M)`
  (success, step 7), `HEARTBEAT_OK` (nothing-to-do reply), and now
  `BLOCKED-TASKS-WIP` (dirty-`tasks/` abort, step 2). The abort path must NOT
  fall through to step 7's `OK` line.
- `$RALPH_HARNESS=claude` this run â†’ `Submitted-by: Claude`.
- **Worktree isolation in `crons/cleanup-tasks.md`**: archive work runs in
  `.worktrees/archive/$TODAY` (branch `archive/$TODAY` off `origin/$BASE`), driven
  by `git -C .worktrees/archive/$TODAY <cmd>` â€” never `cd`. Lifecycle is
  prune-stale-first â†’ `git worktree add` â†’ `trap ... EXIT` (error/abort) +
  explicit step-7 teardown (normal/N=0). The `archive/` subfolder is documented
  in `.worktrees/README.md`'s branch-worktree table.
- **Step numbers are load-bearing**: prose cross-references steps by number
  (e.g. step 2 â†’ "step 7's `OK` line"). Add sub-bullets, not new numbered steps,
  to avoid invalidating those references.

Added `evals/probes/cleanup-tasks-scoped-guard.sh` (executable, mode 100755),

shared-checkout `git switch -c "archive/` is absent (asymmetric oracle). Includes
the required `# tier:`/`# source: issue #85`/`# desc:` headers, `set -euo pipefail`,
and a `# NOTE:` recording the static-grep-not-runtime limitation. Verified PASS
(exit 0) against the current cron and exit 1 on each mutated regression case +
exit 2 on the absent-cron path.

### Learnings for future iterations
